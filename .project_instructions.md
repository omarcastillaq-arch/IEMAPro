# Horizon RPM Mobile App - Project Instructions

## Overview
Remote Patient Monitoring (RPM) mobile app with Spanish UI. Dark theme, medical/health domain.
Two roles: PATIENT (3 tabs: Inicio, Signos, Perfil + capture-vital modal) and DOCTOR (3 tabs: Inicio, Pacientes, Alertas + patient-detail drill-down).

## Tech Stack
- Expo Router (file-based routing), React Native Paper, react-native-chart-kit, AsyncStorage for token
- Axios with JWT interceptor; 401 auto-clears auth

## Theme
- Dark mode: bg #0A0A0A, surface #111111, card #161616
- Primary #4F46E5 (indigo), accent #06B6D4 (cyan), success #10B981, error #EF4444, warning #F59E0B
- Text: primary #F1F5F9, secondary #94A3B8, tertiary #64748B
- Gradient buttons: [#4F46E5, #06B6D4]

## Route Structure
```
app/
  _layout.tsx         → Root: AuthProvider + PaperProvider
  index.tsx           → Redirect based on auth state
  +not-found.tsx
  auth/
    _layout.tsx       → Guard: authenticated → /tabs
    login.tsx
  tabs/
    _layout.tsx       → Guard: !authenticated → /auth/login; role-based tab visibility
    index.tsx         → Dashboard (role-aware: patient or doctor)
    vitals.tsx        → Patient: vitals history with filters + pagination
    profile.tsx       → Patient: profile info + logout
    patients.tsx      → Doctor: patients list with search
    alerts.tsx        → Doctor: alerts with priority/status filters
  capture-vital.tsx   → Patient: record vital (BP, HR, Glucose)
  patient-detail/
    _layout.tsx
    [enrollmentId]/index.tsx → Doctor: detail with vitals/charts/alerts tabs
```

## API Contracts
- Auth: POST /api/auth/login → {token, user}, GET /api/auth/me → {user}
- Patient: GET /api/patients/dashboard → {enrollment, lastVitals, recentVitals, recentAlerts}
- Profile: GET /api/patients/profile
- Vitals: POST /api/vitals, GET /api/vitals/:enrollmentId
- Alerts: GET /api/alerts/:enrollmentId, PUT /api/alerts/:alertId/acknowledge
- Doctor: GET /api/doctor/dashboard, /patients, /patients/:id, /alerts

## Auth Pattern
- Layout-guard with <Redirect>, never imperative navigation after login
- Token in AsyncStorage, auto-attached via axios interceptor
- 401 clears token + sets user=null

## Language
ALL UI text in Spanish (es-MX locale)
